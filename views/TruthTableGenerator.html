Tabela Verdade

Este gerador é capaz de receber uma expressão lógica e retornar a sua respectiva tabela verdade.

Os seguintes caracteres são aceitos pelo gerador:

• Negação: !
• Conjunção: &&
• Disjunção: ||
• Implicação: =>
• Bicondicional: <=>

Lembre-se de utilizar parênteses para delimitar as expressões, como por exemplo:

• !(A || B) <=> (A && C)
• (A => B) && !C
